# Vexa Landing Page — Premium SaaS Design

Este projeto é uma landing page de alta fidelidade para a **Vexa**, uma plataforma de inteligência para gestão de Studios de Beleza e Barbearias.

## 🚀 Tecnologias
- **React 19** + **Vite 8**
- **Tailwind CSS** (Design System Customizado)
- **GSAP** (Animações Cinemáticas e ScrollTrigger)
- **Lucide React** (Ícones)

## 🛠️ Estrutura do Projeto
A estrutura foi otimizada para deploy na **Vercel**, com o código fonte localizado na raiz para facilitar a integração contínua.

## 📦 Como rodar localmente
1. Instale as dependências: `npm install`
2. Rode em modo desenvolvimento: `npm run dev`
3. Gere o build de produção: `npm run build`

## 🌐 Deploy na Vercel
O projeto já conta com o arquivo `vercel.json` configurado para lidar com rotas de Single Page Application (SPA). Basta conectar o repositório à Vercel e o deploy será automático.
